***Current Features:***
- Line numbering
- Terminal
- Find and Replace
- Opening / Saving
- File creating
- Copy & paste
- Clearing text
- Clearing terminal
- Basic python file running
- Syntax highlighting
- Automatic indentation
- Menubar
- Template support
- Settings
- Bug Report
- Autofill/intellisense
- Dark and light mode
- Easy commenting with Command-Slash
- Command-r and/or F5 runs file
- Multi-file support
- Shortcuts window
- Settings function to add keywords to autocomplete
- Update reminder
- Documentation
- Reset Button
- Drag and drop
